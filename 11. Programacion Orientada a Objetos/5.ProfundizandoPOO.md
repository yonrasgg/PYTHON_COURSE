## Definici√≥n de la clase

```python
class FabricaCarros(): # Clase Padre o Super Clase
```

Aqu√≠ estamos definiendo una nueva clase llamada `FabricaCarros`. En la programaci√≥n orientada a objetos, una clase es como un plano que define las caracter√≠sticas y comportamientos que tendr√°n las instancias de esta clase (los objetos).

## El m√©todo constructor `__init__`

```python
def __init__(self, marca, modelo, color, llantas, combustible): # Metodo Constructor
```

El m√©todo `__init__` es un m√©todo especial en Python que se ejecuta cuando se crea una nueva instancia de la clase. Los par√°metros `marca`, `modelo`, `color`, `llantas` y `combustible` se pasan al m√©todo `__init__` y se utilizan para inicializar los atributos de la instancia.

## Atributos de la instancia

```python
self.marca = marca # Atributo de Clase
self.modelo = modelo  # Atributo de Clase
self.color = color  # Atributo de Clase
self.llantas = llantas # Atributo de Clase
self.combustible = combustible  # Atributo de Clase
```

Estas l√≠neas de c√≥digo est√°n inicializando los atributos de la instancia. Cada instancia de `FabricaCarros` tendr√° sus propios atributos `marca`, `modelo`, `color`, `llantas` y `combustible`.

## Creaci√≥n de una instancia

```python
carro = FabricaCarros('Toyota', {'modelo1': 'Corolla', 'modelo2': 'Prius', 'modelo3': 'Celica', 'modelo4': 'Supra', 'modelo5': 'Land Cruiser', 'modelo6': 'FJ Cruiser'}, ['Rojo', 'Azul', 'Verde'], 4, ['Gasolina', 'Diesel', 'Electrico']) # Instancia de la Clase 
```

Esta l√≠nea de c√≥digo est√° creando una nueva instancia de la clase `FabricaCarros`. Los argumentos pasados al constructor (`'Toyota'`, `{'modelo1': 'Corolla', 'modelo2': 'Prius', 'modelo3': 'Celica', 'modelo4': 'Supra', 'modelo5': 'Land Cruiser', 'modelo6': 'FJ Cruiser'}`, `['Rojo', 'Azul', 'Verde']`, `4`, `['Gasolina', 'Diesel', 'Electrico']`) se utilizan para inicializar los atributos de la instancia.

## Impresi√≥n de los atributos de la instancia

```python
print('Marca del Carro: ', carro.marca) # Impresion del Metodo Constructor marca
print('Modelo del Carro: ', carro.modelo) # Impresion del Metodo Constructor modelo
print('Color: ', carro.color) # Impresion del Metodo Constructor color
print('Cantidad de llantas: ', carro.llantas) # Impresion del Metodo Constructor llantas
print('Tipo de combustible: ', carro.combustible) # Impresion del Metodo Constructor combustible
```

Estas l√≠neas de c√≥digo est√°n imprimiendo los atributos de la instancia `carro`. Podemos acceder a los atributos de una instancia utilizando la sintaxis `instancia.atributo`.
