Python tiene varios tipos de datos num√©ricos incorporados. Los m√°s comunes son `int` (enteros) y `float` (n√∫meros de punto flotante).

```python
# Definir un n√∫mero entero
num1 = 40
print(type(num1))  # Imprime: <class 'int'>
print(num1)  # Imprime: 40
```

En este c√≥digo, `num1` es un entero (int), por lo que `type(num1)` devuelve `<class 'int'>`.

```python
# Convertir un n√∫mero entero a float
print(float(num1))  # Imprime: 40.0
```

Aqu√≠ utilizamos la funci√≥n `float()` para convertir el entero `num1` a un n√∫mero de punto flotante, que se imprime como `40.0`.

```python
# Definir un n√∫mero de punto flotante
num2 = 99.99
print(type(num2))  # Imprime: <class 'float'>
print(num2)  # Imprime: 99.99
```

En este c√≥digo, `num2` es un n√∫mero de punto flotante (float), por lo que `type(num2)` devuelve `<class 'float'>`.

```python
# Convertir un n√∫mero de punto flotante a int
print(int(num2))  # Imprime: 99
```

Aqu√≠ utilizamos la funci√≥n `int()` para convertir el n√∫mero de punto flotante `num2` a un entero. Ten en cuenta que esto trunca el n√∫mero hacia cero, descartando cualquier cosa despu√©s del punto decimal.

Estos son los tipos de datos num√©ricos m√°s com√∫nmente utilizados en Python. Python tambi√©n soporta otros tipos de datos num√©ricos, como los n√∫meros complejos (`complex`), pero estos son menos comunes.
